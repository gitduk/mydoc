Git Tutorial

文章适合

获取帮助

man git-log
# or
git help log

设置

# set editor to vim
git config --global core.editor "vim"

# list all configs of global
git config --list

 

 

创建一个储存库

cd project_dir
git init

 

添加文件到暂存区

git add file1 file2 file3 ...
# or
git add .

 

提交修改到版本库

git commit -m "description"

 

查看版本库状态

git status

 

查看已经提价的修改

git diff --cached

 

提交一步到位

git commit -a
